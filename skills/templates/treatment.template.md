## TREATMENT CHECK

After filling in, walk every scene and answer:

1. **Does each scene start with "because"?** (Scene N happened because in scene N-1, X happened.)
2. **Does each scene move value** from + to – or vice versa?
3. **Does each scene serve the controlling idea?**
4. **If you remove the scene, does the film change?** (If not — extraneous.)

If even one answer is "no," rewrite or remove the scene.

---

## AUDIT TAGS

When you find a problem, mark it:

- ⚠ **[CAUSALITY]** — scene doesn't follow from the previous one
- ⚠ **[VALUE]** — no value movement
- ⚠ **[BIBLE]** — conflict with character or world bible
- ⚠ **[TEMPO]** — too fast / slow for its function
- ⚠ **[INFO DUMP]** — exposition through dialogue
- ⚠ **[REPEAT]** — repeats something already established
- ⚠ **[FLAW AMNESIA]** — the hamartia is forgotten in this scene
