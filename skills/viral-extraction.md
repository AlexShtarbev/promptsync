# Viral Extraction & Distribution

How to extract short clips from a locked storyboard, build a distribution campaign, and design the funnel from shorts to the full piece.

This file is read after `short-form.md` and `timing-and-cutting.md`. It assumes the main storyboard is locked and image prompts exist.

---

## WHEN TO USE

Trigger when the user says any of: "extract a clip," "viral short," "make shorts from this," "distribution plan," "how do I promote this," "campaign," "funnel," "tease," "trailer," "cut a 15s / 30s from this," or asks how to drive traffic to the full piece.

Do not use before the main storyboard is locked. Extraction depends on finished shot IDs, durations, and asset mapping.

---

## PRINCIPLE: EXTRACT, DON'T COMPRESS

A viral clip is not the full piece made shorter. It is a **self-contained story that already lives inside the material.** The job is to find it, not to force it.

Three rules:

1. **Every clip must have its own Promise / Progress / Payoff.** If you can't name all three in one sentence each, the clip isn't a story — it's a fragment. Fragments don't hold attention.
2. **Every clip must work standalone.** A viewer who never sees the full piece should still get a complete experience. Mystery is fine. Confusion is not.
3. **No clip should spoil the full piece's payoff.** The clip can show the question. It should not show the answer — or it should reframe the answer so the full piece still adds a layer.

## THE EXTRACTION METHOD

### Step 1 — Map the emotional arc

Before looking for clips, write the full piece's emotional arc as a sequence of one-word states. Example: *effort → recognition → hope → loss → emptiness → will → defiance → dignity.*

Each word is a potential entry point or exit point for a clip.

### Step 2 — Identify self-contained arcs within the sequence

Look for subsequences that form their own P/P/P:

- **A single turn** (2–3 states): hope → loss. That's a 15s gut-punch.
- **A rise arc** (3–4 states): emptiness → will → defiance. That's a 15–19s statement.
- **A fall-and-rise** (4–6 states): hope → loss → emptiness → will → return. That's a 30s story.
- **A full mini-arc** (6+ states): needs significant restructuring and is likely just the full piece minus the opening. Only attempt if the piece is over 90s and has clearly separable layers.

Each subsequence must start and end on a state the viewer can enter and exit without context. "Hope → loss" works cold. "Recognition → hope" requires setup that isn't in the clip.

### Step 3 — Map subsequences to shots

For each candidate arc, pull the specific shot IDs from the main storyboard. Reference them by ID (e.g. "= 3C from main") so every clip maps directly to existing generated assets.

Check:
- [ ] Does the clip use only shots that are already in the main storyboard?
- [ ] Does the clip require zero additional asset generation? (Ideal. If new shots are needed, flag it.)
- [ ] Do the shots work in this new order without visual continuity breaks? (Watch for: palette shifts that need the main piece's setup, characters who appear without introduction, objects that are payoffs without their setup.)

### Step 4 — Solve the hook

Every clip needs its own hook in the first 1–2 seconds. The main piece's hook may not work as a clip hook (it may be too slow, or it may depend on preceding context).

For each clip, ask: **"If this played on autoplay in a feed with no context, would the first frame + first sound stop a thumb?"**

If not, consider:
- Starting the clip mid-action (enter the scene already in motion).
- Using the strongest audio moment as the opener (effort sounds, a sharp cut from silence, a single VO line).
- Opening on the most visually arresting frame, even if it's not the first beat chronologically.

### Step 5 — Solve the ending

Each clip needs a deliberate ending pattern (see `short-form.md` for the full list). The two most effective for extraction clips:

- **Hard Cut to Black** — works for gut-punches, falls, unresolved tension. The absence of resolution is the hook to the full piece.
- **Emotional Linger** — works for statements of will, resolve, defiance. Hold the final image 1–2s past the last sound. The stillness is the payoff.

A third option specific to funnel clips:

- **Truncated VO** — cut a VO line mid-sentence. The missing word creates a completion gap. The viewer seeks the full piece to hear the rest. Use sparingly — once per campaign maximum, or it feels like a gimmick.

### Step 6 — Check for overexposure

If you're extracting multiple clips from one piece, map which shots appear in which clips. The rule: **no shot should appear in more than two clips** (including the full piece). If the same frame appears in three shorts and the main video, the viewer has seen it four times before they've watched the full piece. The payoff is dead.

The most powerful shots — the ones that carry the climax or the emotional peak — should appear in **one clip only**, plus the full piece. Reserve them.

## CAMPAIGN ARCHITECTURE

### How many clips?

- **1 clip** — a trailer. Good for a single push, not a campaign.
- **2–3 clips** — standard extraction. One hook, one emotional beat, one payoff tease.
- **5 clips** — a campaign. Each clip reveals one layer. This is the maximum before overexposure risk becomes unmanageable.
- **More than 5** — the piece probably isn't long enough to support it, or the clips will overlap too much. Push back.

### Sequencing

Order clips by **what they reveal**, not by their chronological position in the full piece:

1. **First clip = the world.** Introduce the visual and tonal identity. No resolution. Pure hook.
2. **Middle clips = the layers.** Each adds something the previous one didn't: a new character, a new emotion, a new question. Escalate stakes or deepen the metaphor.
3. **Second-to-last clip = the first complete arc.** This is the first time a viewer gets a satisfying ending from a clip. Reward the ones who've been following.
4. **Final clip = the peak image / statement.** The most intense, most shareable, most emotionally concentrated extraction. This is the one that gets screenshotted, duetted, quote-tweeted. Reserve it.
5. **Full piece = the answer.** Everything the clips promised, delivered in full.

### Release cadence

| Cadence | When to use | Risk |
|---|---|---|
| **Daily over N days** (full piece on final day) | Standard campaign. Gives each clip room to breathe. | If one clip underperforms, momentum stalls. |
| **Skip a day before the longest clip** | If you have 3+ short clips followed by a 30s clip. The gap creates anticipation. | The gap must be earned — only skip if the first clips are performing. |
| **Two clips in one day** (4–6 hours apart) | Compressed cadence. Good for impatient audiences or platforms that reward frequency. | Audience may only see one of the two. |
| **All clips + full piece same day** | No campaign, just a content drop. | No funnel effect. Each clip competes with the others for attention. |

### Funnel mechanics

The goal is: **clip → profile visit → full piece → follow.**

- Do not put "watch the full video" in the clip. It's desperate and it suppresses reach on most platforms.
- Pin a comment on each clip: the title of the full piece, nothing else. Or "tomorrow" if the full piece drops the next day.
- Use a consistent audio element across clips. The algorithm and the viewer's ear link them. Name it as an original sound.
- Caption each clip with a line from the piece itself — VO, a key phrase, a question. Do not explain the clip in the caption.

## THUMBNAIL STRATEGY

### Principles

- One focal point per thumbnail. No collage.
- Human element: face, hands, or eyes. Thumbnails with human features outperform landscapes.
- Each clip in a campaign should have a **visually distinct** thumbnail. When viewed together on a profile grid, the set should tell the story in still images.
- Do not spoil the peak image in an early thumbnail. The most intense frame is reserved for the final clip.

### Assignment method

For each clip, choose the thumbnail from the clip's shots by asking: **"Which single frame from this clip would make me tap?"** Priority order:

1. A face with an unresolved expression (eyes, tension, ambiguity).
2. An object that tells a story (a shoe, an empty hand, a tool at rest).
3. A composition with dramatic scale (figure vs landscape, extreme close-up vs wide).
4. A palette break (if the campaign is mostly cold, the one warm thumbnail stands out).

### Text on thumbnails

Default: no text. Let the image work. If text is added:
- Shorts: no text overlay on thumbnail. The platform title is enough.
- Full piece: project title only, lower third, clean font, no tagline.

## STORYBOARD OUTPUT FOR CLIPS

Each extracted clip gets its own TSV storyboard using the same column format as the main storyboard (see `storyboard.md` for the full TSV column spec). Additional requirements:

- Every shot row must reference the main storyboard shot ID in the Notes column: `= [ID] from main.`
- If a shot is trimmed from the main version, note the new duration and what was cut: `= 1A from main. Trimmed from 6s to 4s — enter mid-push.`
- If a shot's VO is truncated or split differently than the main, note the exact VO used in this clip.
- The TSV header row should include the clip title and duration: use a metadata row or a clear header comment.

No new image prompts are needed unless a clip requires a shot that doesn't exist in the main storyboard. If new shots are needed, flag them explicitly and write prompts following the format in the main image prompts file.

## DISTRIBUTION STRATEGY DOCUMENT

When the user asks for a distribution plan, produce a single markdown file covering:

1. **The campaign map** — which clips, what order, what each reveals.
2. **Posting cadence** — day-by-day schedule with rationale. Include an alternative compressed cadence.
3. **Platform-specific notes** — format, features (series, playlists, sounds), and algorithm considerations for each target platform.
4. **Thumbnails** — one per clip + full piece, with the source shot ID and why it was chosen.
5. **Captions** — one line per clip. Default: use lines from the piece itself.
6. **Audio strategy** — original sound naming, consistent SFX, how the audio links the campaign.
7. **Engagement rules** — what to do and what not to do in comments and replies.
8. **Metrics** — what to watch per clip, what targets to aim for, and what to do if a clip underperforms.
9. **Post-campaign** — playlist/series creation, profile pinning, evergreen strategy, sequel potential.

## WHAT THIS FILE DOES NOT DO

1. Does not extract clips before the main storyboard is locked.
2. Does not create clips that require assets not in the main storyboard (unless explicitly flagged and approved).
3. Does not compress the full piece into a shorter version. Extraction ≠ compression.
4. Does not recommend more than 5 clips from a single piece.
5. Does not put "link in bio" or "watch full video" in captions or overlays.
6. Does not use the peak image / payoff shot in more than one clip.
